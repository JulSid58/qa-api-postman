# API Testing with Postman

This repository contains API test collection created with Postman.

## 🧪 Coverage
- Authentication (sign in / sign up)
- Positive scenarios
- Negative scenarios
- Response validation

## 🛠 Tools
- Postman
- JavaScript (Tests tab)

## ▶️ How to use

1. Import collection into Postman
2. Run via Collection Runner
